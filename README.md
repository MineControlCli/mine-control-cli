<h1 align="center">
Mine Control CLI
</h1>
<p align="center">  
  A command-line interface (CLI) application for managing Minecraft servers.
</p>

## Technologies

- Java 21
- Spring Boot
- Spring Shell

## Features

- Create a new Minecraft server
- List all available servers
- Delete a server
- Start a server
- List all server loaders
- Backup a server
- Exit the application
